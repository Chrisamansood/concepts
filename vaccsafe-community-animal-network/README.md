# VaccSafe Community Animal Network

An interactive concept walkthrough for QRB's community animal identity and care platform.

The standalone prototype demonstrates:

- Public animal discovery and approximate-location mapping
- Rapid animal sighting and help-report workflows
- QR-based profile retrieval
- Persistent animal identity and chronological care history
- Searchable animal records
- Admin case assignment and resolution
- Vaccination, care-outcome, and data-quality insights
- A dedicated developer handoff covering architecture, Google Sheets structure, stack, cost, and delivery phases

## Open locally

Open `index.html` in a browser. The prototype is entirely front-end and uses fictional demonstration data. Animal photographs are loaded from Unsplash and therefore require an internet connection.

## Proposed pilot stack

- Google Sheets
- Google Apps Script
- Google Drive
- HTML, CSS, and JavaScript
- MapLibre or Leaflet with OpenStreetMap
- Chart.js
- Looker Studio
